# Contact Form with JavaScript Validation

## Objective
Build a simple contact form with client-side validation using JavaScript.

## Features
- Input fields: Name, Email, Message
- Clean and modern user interface
- Client-side validation using JavaScript
- Regex-based email validation
- Error messages displayed below each input field
- Prevents form submission if validation fails
- Displays success message when form is valid
- Handles edge cases like empty inputs and invalid email format

## Technologies Used
- HTML
- CSS
- JavaScript

## Validation Rules
- Name cannot be empty
- Email must follow valid email format
- Message cannot be empty
- Form submission is prevented if inputs are invalid

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in a browser
3. Test the form with valid and invalid inputs

## Learning Outcome
- Understanding form handling in JavaScript
- DOM manipulation
- Event handling
- Regular expression (regex) validation
- Displaying dynamic user feedback
